# Анализ сырых данных из Pure API

## Описание

Данный репозиторий содержит код, написанный в ходе изучения сырых данных, получаемых от сервися Pure API.

Основная цель - изучить структуру сущностей сервиса, выявить первые проблемы работы с ними.
Это позволит определить дальнейший план действий.

Весь отчет (текст + код) представлен в `notebooks/explore.ipynb`.
Дополнительные функции для переиспользования размещены в `notebooks/json_utils.py`.

Исходные данные можно получить в репозитории проекта.
Скачать и распаковать в папку `data`.

В `bibliography` находится доп.литература для дальнейшего ознакомления.

## Запуск и получение артефактов

Для запуска проекта необходимо установить UV.
Инструкции по установке доступны по [ссылке](https://docs.astral.sh/uv/getting-started/installation/#installation-methods).

Создать виртуальное окружение:

```bash
uv venv
```

Активировать виртуальное окружение:

```bash
source .venv/bin/activate # Linux/Mac

.venv\Scripts\activate # Windows
```

Установить зависимости:

```bash
uv pip install -e .
```

Запуск Jupyter Notebook может производиться двумя способами:

1. Используя Jupyter Notebook в браузере:

```bash
uv run jupyter lab
```

2. Используя VSCode (с расширением): выбрать ядром (select kernel) ядро из `venv`
